Authenticator.Spree Class

Extends SimpleAuthDevise.Authenticator

Defined in: addon/authenticators/spree.js:2

The Spree Authenticator is responsible for Authenticating users against your Spree store. It assumes your server has the spree_ams gem installed. The Spree Auth initializer with dynamically create and set the Server Token Endpoint from the Spree Core Adapter.
